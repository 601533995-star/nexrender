## 🚀 Quick Start (Commercial Version)

### Installation
```bash
git clone https://github.com/601533995-star/nexrender.git
cd nexrender
npm install
```

### License Configuration
```javascript
const { NexRenderPremium } = require('./commercial/premium-features');

// Initialize with license key
const premium = new NexRenderPremium('your-license-key-here');

// Check feature availability
if (premium.checkPermission(['pro', 'enterprise'])) {
    // Access premium features
    const analytics = premium.advancedAnalytics();
    const cloudStorage = premium.cloudStorage();
}
```

### Professional UI Setup
```html
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="commercial/professional.css">
</head>
<body>
    <div class="nexrender-dashboard">
        <!-- Professional UI components -->
    </div>
    <script src="commercial/premium-features.js"></script>
</body>
</html>
```
